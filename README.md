## Overview
The game centers around fireboy and watergirl naviagting through various levels in order to get to their respective exit doors. Fireboy is allergic to blue water, and must collect red gems, while watergirl is allergic to red water and must collect the blue gems. The players must work together to solve puzzles using sliding doors and parkour through the platforms.  
## Instructions
Watergirl can be moved left and right with the A and D keys and can jump with the W key. Fireboy moves with the arrow keys. The game can be paused and the pause menu can be pulled up by clicking ESC. Also, gamestates can be cycled by clicking the L key (this is a cheat code).
