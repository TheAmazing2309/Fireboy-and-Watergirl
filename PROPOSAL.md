# IDEA:

 #### Video player in the terminal
 This program would take a mp4 file and break it down into its individual frames and print the frames one at a time in the terminal using colored ASCII characters. The program would optimize the frame rate by grouping pixels that are similar in color.

 #### Sort visualizer
 This program would consist of different sorting algorithms and display the current actions and movements of the items as the code is running. There would be a menu to add optional modifiers to the algorithms to show how those work as well.

 #### Fire boy and water girl game
 This program is a replica of the game Fire boy and Water girl. We will make maps by using text files that have key words which will be read by our program to form a map to play on.
